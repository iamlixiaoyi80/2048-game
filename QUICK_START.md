# 弹幕肉鸽游戏 - 快速开始

## 🚀 方式1: 本地运行（推荐）

### 步骤1: 克隆仓库

```bash
git clone https://github.com/iamlixiaoyi80/2048-game.git
cd 2048-game
```

### 步骤2: 安装依赖

```bash
npm install
```

### 步骤3: 启动开发服务器

```bash
npm run dev
```

### 步骤4: 访问游戏

在浏览器中打开: `http://localhost:8080`

## 📋 游戏功能

### MVP版本（第一阶段）

✅ **登录系统**
- 竖版界面（9:16）
- 账号密码输入
- 新账号/老账号检测
- localStorage 存档
- Q版卡通风格（金色按钮）

✅ **主界面**
- 游戏标题
- 新游戏按钮
- 继续游戏按钮
- 返回按钮
- Q版卡通风格（金色按钮）

✅ **人物选择系统**
- 3 个人物选择（狙击手、重装兵、刺客）
- 人物详情显示（种族、职业、天赋、属性）
- 可滑动列表
- Q版卡通风格（卡片布局）

✅ **关卡选择系统**
- 3 个关卡选择（新手村、魔法森林、沙漠遗迹）
- 关卡信息显示
- Q版卡通风格（卡片布局）

## 🎮 操作说明

### 登录系统
1. 输入账号（任意字符）
2. 输入密码（任意字符）
3. 点击"登录"按钮
4. 新账号 → 进入主界面 → 点击"新游戏"
5. 老账号 → 进入主界面 → 点击"继续游戏"

### 人物选择
1. 点击人物卡片查看详情
2. 点击"选择"按钮选择人物
3. 进入关卡选择界面

### 关卡选择
1. 点击关卡卡片查看详情
2. 点击"开始"按钮开始关卡（TODO）
