## 🌐 在线体验（立即访问！）

### 🥇 方式 1: 直接访问（推荐）

点击以下链接，游戏会直接在浏览器中运行：

**游戏地址**：
```
https://raw.githubusercontent.com/iamlixiaoyi80/2048-game/main/index.html
```

### 🥈 方式 2: GitHub + Raw 按钮

1. 访问：https://github.com/iamlixiaoyi80/2048-game/blob/main/index.html
2. 点击右上角的 **"Raw"** 按钮
3. 游戏会直接在浏览器中运行

### 🥉 方式 3: 本地运行

```bash
git clone https://github.com/iamlixiaoyi80/2048-game.git
cd 2048-game
npm install
npm run dev
```

然后访问: `http://localhost:8080`

## 📖 游戏功能

### MVP 第一阶段（已完成）✅

- ✅ **登录系统** - 账号密码登录 + localStorage 存档
- ✅ **主界面** - 新游戏/继续游戏
- ✅ **人物选择** - 3个人物（狙击手、重装兵、刺客）
- ✅ **关卡选择** - 3个关卡（新手村外围、魔法森林、沙漠遗迹）
- ✅ **Q版风格** - 竖版手机界面，卡通设计
